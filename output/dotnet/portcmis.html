Welcome to PortCMIS¶

Apache Chemistry PortCMIS is a CMIS 1.0 and CMIS 1.1 client library for .NET. It is loosely based on the OpenCMIS client architecture.

PortCMIS is the successor of DotCMIS.

Source code, dependencies and development binaries¶

The PortCMIS source code is here: https://svn.apache.org/repos/asf/chemistry/portcmis/trunk/.

PortCMIS requires a .NET Standard 1.1 runtime or higher. It does not depend on anything else.
